# ✊✋✌ Rock Paper Scissors Game

An interactive and fun Rock Paper Scissors game built using HTML, CSS, and JavaScript.
The player competes against the computer with real-time result display and score tracking.

## 📌 Features

- ✅ Player vs Computer gameplay

- ✅ Random computer choice generation

- ✅ Real-time result display (Win / Lose / Draw)

- ✅ Score tracking system

- ✅ Responsive and clean UI

- ✅ Instant game reset functionality


## 🛠️ Technologies Used

- HTML5 – Game structure

- CSS3 – Styling and layout

- JavaScript (ES6) – Game logic and DOM manipulation

## 🎮 How the Game Works

1. The player selects Rock, Paper, or Scissors.

2. The computer randomly selects one option.

3. The game compares both choices:

4. Rock beats Scissors

5. Scissors beats Paper

6. Paper beats Rock

7. The result is displayed instantly.

8. Scores are updated accordingly.

## 🧠 Game Logic

- Used JavaScript Math.random() for computer choice.

- Used conditional statements to determine winner.

- DOM manipulation updates score and result dynamically.
